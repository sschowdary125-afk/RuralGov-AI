# 🏛 RuralGov AI

## AI-Powered Government Scheme Eligibility & Application Readiness Assistant


## 🚀 Overview

RuralGov AI is an AI-powered platform that helps citizens understand their eligibility and application readiness before applying for government schemes.

The platform analyzes applicant information, identifies missing documents, predicts application approval probability, highlights possible risks, and provides personalized recommendations using Generative AI.


---

# 🎯 Problem Statement

Millions of citizens face difficulties while applying for government schemes because of:

- Lack of awareness about suitable schemes
- Confusion regarding eligibility criteria
- Missing required documents
- Incorrect application preparation
- Increased chances of application rejection due to incomplete information


---

# 💡 Solution

RuralGov AI acts as a digital government assistance companion that helps citizens prepare better applications.

The system provides:

✅ Eligibility analysis  
✅ Approval prediction  
✅ Missing document identification  
✅ Risk detection  
✅ Personalized recommendations  


# ✨ Key Features


## 1. Government Scheme Discovery

Users can explore and analyze different government schemes:

- PM-KISAN
- PMAY
- Ayushman Bharat

- Sukanya Samriddhi Yojana


## 2. Smart Eligibility Assessment

Users provide:

- Personal information

- Location information
- Available documents
- Scheme-specific information


## 3. AI-Based Application Analysis

Powered by Google Gemini AI.

The AI evaluates:

- Eligibility score
- Approval prediction
- Missing documents
- Application risks
- Improvement recommendations


## 4. Document Readiness Verification

The system compares:

User-provided documents

with

Scheme-specific required documents

and identifies missing requirements before submission.

# 🤖 AI Architecture

The application follows this AI workflow:

1. User submits application details
2. React collects and processes user inputs
3. Scheme requirements are matched dynamically
4. Google Gemini AI analyzes eligibility
5. AI generates structured recommendations
6. Dashboard displays actionable insights
